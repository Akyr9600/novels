# よく使うMarkdown記法

## 文字の装飾

| 見た目 | 書き方 |
|--------|--------|
| **太字** | `**テキスト**` |
| *斜体* | `*テキスト*` |
| ***太字＋斜体*** | `***テキスト***` |

---

## 見出し

```
# 見出し1（最大）
## 見出し2
### 見出し3
```

---

## 区切り線（場面転換）

```
---
```

3つのハイフンで区切り線になる。テーマCSSの色が反映される。

---

## Hugo Shortcode：歌詞カード

歌詞や詩を「カード」として表示したいとき：

```
{{</* lyric */>}}
"曲のタイトル"

1行目の歌詞
2行目の歌詞

3行目の歌詞
{{</* /lyric */>}}
```

中身は斜体・中央揃えで表示される。

---

## その他メモ

- 改行1つ → Hugoが同じ段落としてまとめる（くっつく）
- 改行2つ（空行あり）→ 別の段落になる
- 場面転換の大きな空白は `---` を使う

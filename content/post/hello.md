## ■ 基本コマンド
これらは「サイトの一番上の階層（hugo.tomlがある場所）」で実行すること。

| コマンド | 辞書登録名 | 意味 |
| :--- | :--- | :--- |
| hugo server -D | ぱわ | 下書きも含めてローカルでプレビュー（基本これ） |
| Ctrl + C | (なし) | プレビューサーバーを停止する |

---

## ■ フロントマター（名札）の書き方
各 .md ファイルの冒頭に書く「サイト上の振る舞い」を決める要素。

---
title: "お話のタイトル"
weight: 10
draft: true
---

### ■ 各要素の意味
- title: 目次やページ上部に表示される名前。
- weight: 並び順。数値が小さいほど上（手前）に来る。10刻み推奨。
  - _index.md に書けば、シリーズ自体の並び順も変えられる。
- draft: 公開設定。
  - true: プレビューでは見えるが、本番サイトではリンクが消える（今回の仕様）。
  - false または 項目ごと削除: 本番サイトで公開される。
- coming-soon: true
　- 「準備中」としてタイトルを見せつつクリック不可にする
- default_group: "フィルとちびこ"
　- フォルダ内のメインの区分。これが「記入なし」の区分として書かれる
- group: "リュシアン"
　- フォルダ内の他の区分。書かれていると、この名前で分けられる
---

## ■ フォルダ構成と役割
- content/: 小説本文（.md）を入れる場所。ここをいじればサイトが更新される。
- layouts/: サイトの見た目（HTML/CSS）を決める場所。
- hugo.toml: サイト全体の設計図（タイトル設定など）。

---

## ■ 運用メモ
- 「準備中」の作り方:
  まだ中身を見せたくないお話には draft: true をつける。すると目次にはタイトルだけ出て、クリックできない状態になる。
- リライト時:
  公開済みのお話を大幅に直す時は、一時的に draft: true を書き足せば、読者からは中身が見えなくなる。
- 自分用メモ:
  このファイルのように draft: true にしたものは、自分だけがローカル環境で見られる「秘密のメモ」として使える。
